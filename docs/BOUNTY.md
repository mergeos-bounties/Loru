# Loru bounties (MergeOS MRG)

Loru pays contributors in **MRG** through the MergeOS bounty program.

## Before you start

1. Star https://github.com/mergeos-bounties/Loru  
2. Star https://github.com/mergeos-bounties/mergeos  
3. Comment on the Loru issue: `I claim this bounty`  
4. Comment on MergeOS [Claim Token #1](https://github.com/mergeos-bounties/mergeos/issues/1) with the issue link  
5. Open a PR to **Loru** `master` (`Fixes #N`)

## Payout scale

| Scope | Typical MRG |
| --- | ---: |
| Small / good-first | 25 |
| Medium feature | 50 |
| Large system piece | 100 |
| Extra-large | 200 |

Issue titles may show marketing amounts; **admin ledger pays 25/50/100/200** after review.

## Domains we fund

- Dataset loaders & annotation formats  
- Landmark extraction (MediaPipe / OpenPose adapters)  
- Training loops & metrics  
- Sign → text inference quality  
- Sign → voice (TTS) pipeline  
- API / realtime webcam demo  
- Evaluation benchmarks & accessibility docs  

## Evidence

- Tests + command output for backend/ML  
- Screenshots or short clips for UI/webcam demos  
- Dataset license notes for any new data
